# mk-cctv-guide

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅用于存放静态页面文件，方便浏览和分享。

## 项目简介

该仓库旨在系统性地整理和发布多个独立的 HTML 页面。每个页面均为独立的静态文件，不依赖后端服务，可直接在浏览器中打开查看。仓库内容主要涉及信息展示、工具页面或说明文档等类型。

## 目录说明

- `pages/` — 存放所有归档的 HTML 页面文件，按功能或主题分类。
- `assets/` — 存放页面所需的静态资源，如图片、样式表、脚本等。
- `docs/` — 存放附加说明文档或参考资料。
- `index.html` — 仓库首页入口，展示页面列表或导航链接。

## 页面归档说明

- 每个 HTML 页面均为独立文件，命名采用有意义的英文或拼音。
- 页面内容不包含跟踪代码、广告、恶意脚本或外部资源依赖。
- 归档页面会定期更新，新增页面将按顺序添加至 `pages/` 目录。
- 旧版页面可能被替换或移除，请以最新版本为准。

## 维护说明

- 本仓库由个人维护，不接受外部贡献或 pull request。
- 如发现页面内容错误或链接失效，可通过 Issues 反馈。
- 仓库内容仅供学习参考，请勿用于商业用途或违法用途。
